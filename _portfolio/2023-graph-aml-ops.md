### Overview
- Built a heterogeneous graph neural network that links transactions, entities, and behavioral features across eight data sources.
- Added counterfactual explanation tooling so investigators can see the minimal feature changes that flip a risk decision.
- Reduced false positives by 19% while speeding investigator triage by 32% in pilot markets.

### Technology Highlights
- Graph attention networks optimized for multi-relational financial data.
- Counterfactual explanation service published through internal APIs and analyst dashboards.
- Automated fairness and drift monitoring with monthly governance reports for regulators.
